# qiwo-im-core

Shared Qiwo input method core modules.

## Modules

- `qiwo-sync-core`: shared Rime/WebDAV sync implementation.
- `qiwo-input-format-core`: shared commit text formatting implementation.

## Setup

```sh
git clone --recurse-submodules https://github.com/LeaWron/qiwo-im-core.git
```

For an existing checkout:

```sh
git submodule update --init --recursive
```
